## 開発のすすめ方
* 2つの Google ハングアウトで行う

    * mint cafe -- mintsns の関わるメンバー全員がjoinしているハングアウト
    * mint develop -- mintsnsの開発者のみのハングアウト
    * mint design -- デザイン専用
    * mint active -- mintsnsで今いる人だけの部屋（一時部屋）
